Nvidia graphic card setting in arch

Posted on 2019-08-13

Two methods are used to switch the grapic card: Bumblebee or Prime.

Bumblebee:

Advantage: Better performance, Easy to setting
Disadvantage: Not support anymore

Prime:

Advantage: Easy to setting
Disadvantage: Performance is not very good

Bumblebee Setting:

  1. Install bumblebee, mesa, nvidia driver and xf86-video-intel(intel Xorg driver)
    If 32-bit application support is needed, lib32-nvidia-utils is required.

  2. Add regular user to the bumblebee group and enable bumblebeed.service
